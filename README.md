# My little disassemble of the [loliland](https://loliland.ru "site") project

## Folder: scripts, Contains a scripts for api reversing

## Folder: launcher, Contains data obtained by reversing the launcher

* **api.txt - contains a api domains**
* **api_paths.txt - contains a api paths**
* **compatibility.txt - contains data about compatibility of launchers in different systems**

## Folder: images, Contains just pictures of our progress

* **mindmap.png - a plan on how to bypass the Launcher's security**
* **anticheat_bypass1.jpeg to confirm that we've bypassed the anticheat (1.12.2)**
* **anticheat_bypass2.jpeg same as first, but other hacked client (1.12.2)**
* **devapi.jpeg contain a dev server status**

## Folder: lolimod, Contains data obtained by reversing the launcher (same as launcher folder)

* **lolimod-deobf.jar - deobfuscated lolimod, obfuscator is: [Superblaubeere](https://github.com/superblaubeere27/obfuscator)**
* **info.txt - information about lolimod**
